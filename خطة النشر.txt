2️⃣ إعداد قاعدة البيانات للإنتاج
الجداول الأساسية

النظام يحتاج هذه الجداول فقط في هذه المرحلة:

3️⃣ إعداد Indexes (مهم للأداء)
create index idx_orders_user
on orders(user_id);

create index idx_tracking_order
on order_tracking_events(order_id);

create index idx_tracking_created
on order_tracking_events(order_id, created_at);

create index idx_cart_items_cart
on cart_items(cart_id);

هذه الفهارس تمنع بطء الاستعلام عندما يكبر النظام.

4️⃣ إعداد RLS (الأمان)
Orders

المستخدم يرى طلباته فقط.

create policy "users see own orders"
on orders
for select
using (auth.uid() = user_id);
Tracking Events
create policy "users see tracking"
on order_tracking_events
for select
using (
order_id in (
select id from orders
where user_id = auth.uid()
)
);

Realtime يجب أن يعمل على:

6️⃣ حماية API

if (!user) {
  return NextResponse.json({ error: "Unauthorized" }, { status: 401 });
}
